Cornerstone Delaware Tamil Prayer Fellowship Website

Overview:
Website for the Delaware campus of Cornerstone Tamil Prayer Fellowship. Provides info on the church, beliefs, ministries, giving, and contact details.

Features:

Responsive design with mobile-friendly hamburger menu.

Pages:

Welcome: Intro + Google Calendar

Who We Are: Church history, pastor, mission

What We Believe: Core doctrines

Giving: Online donations

Contact: Delaware campus info + map

Dynamic footer year

Google Maps & Calendar embeds

Technologies:
HTML, CSS, JavaScript, Google Fonts, Google Maps & Calendar

File Structure:

/ (root)
│── index.html
│── who-we-are.html
│── what-we-believe.html
│── giving.html
│── contact.html
│── css/
│    └── styles.css
│── script.js


Usage:
Open index.html in a browser. Ensure css/styles.css and script.js are in the correct folders.

Customization:
Update campus info in HTML, modify styles in CSS, or update the Google Calendar embed as needed.
