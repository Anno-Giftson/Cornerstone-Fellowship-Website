Cornerstone Delaware Website

Overview:
Website for the Delaware campus of Cornerstone Tamil Prayer Fellowship. Provides info on the church, beliefs, ministries, giving, videos, and contact details.

Features:

Responsive design with mobile-friendly hamburger menu

Dynamic footer year

Google Maps & Calendar embeds

Embedded videos (latest + video grid)

Pages:

Welcome: Intro + Google Calendar

Who We Are: Church history, pastor, mission

What We Believe: Core doctrines

Giving: Online donations

Videos: Embedded latest and other videos

Contact: Delaware campus info + map

Technologies:
HTML, CSS, JavaScript, Google Fonts, Google Maps & Calendar

File Structure:

/ (root)
│── index.html
│── who-we-are.html
│── what-we-believe.html
│── giving.html
│── videos.html
│── contact.html
│── css/
│    └── styles.css
│── script.js
│── videos.js


Usage:
Open index.html in a browser. Ensure css/styles.css, script.js, and videos.js are in the correct folders.

Customization:
Update campus info in HTML, modify styles in CSS, update the Google Calendar embed in the HTML files, or add/remove the number of videos in videos.js as needed.
